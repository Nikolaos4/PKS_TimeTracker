# TimeTracker

Упрощённая версия тайм-трекера для учёта времени задач.  
Проект иллюстрирует применение паттернов Singleton, Facade, Observer и настройку сборки CMake.

## Сборка

1. Установите CMake (version >= 3.10).
2. Откройте терминал в папке проекта.
3. Выполните команды:
    * cmake -S . -B build
    * cmake --build build
4. Запустите: ./build/timetracker


## Используемые библиотеки

Внешних библиотек не требуется. Используется только стандартная библиотека C++.

## Примечание

Все методы — заглушки, демонстрирующие структуру и взаимодействие классов.
